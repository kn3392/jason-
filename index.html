Javascript Datatype:

Example 1 :

true + false

Answer or Error: 1

Why : true is converted to 1, false is converted to 0, so 1 + 0 = 1

Example 2 :

null + 1

Answer or Error: 1

Why : null is converted to 0, so 0 + 1 = 1

Example 3 :

"12" + 12

Answer or Error: "1212"

Why : when one operand is a string, the other is converted to a string, so "12" + "12" = "1212"

Example 4 :

!! undefined

Answer or Error: false

Why : undefined is a falsy value, so !!undefined = false

Example 5 :

"12" - 1

Answer or Error: 11

Why : when one operand is a number, the other is converted to a number, so "12" - 1 = 12 - 1 = 11
